# add TikDown tap
```
brew tap Tairraos/tikdown
```

# install TikDown
```
brew install --cask tikdown
```

# uninstall TikDown
```
brew uninstall --cask tikdown
```

or

# uninstall TikDown with zap
```
brew uninstall --cask tikdown --zap
```

# remove TikDown tap
```
brew untap Tairraos/tikdown
```
